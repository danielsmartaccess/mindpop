# LifeSync - Organize Your World

Based on the Shark Tank pitch, LifeSync is the ultimate organization tool for high schoolers and young professionals.

## Features

*   **Dashboard**: Overview of your day, balance, and mood.
*   **Calendar**: Visual monthly planner.
*   **To-Do List**: Manage your daily grind.
*   **Finance**: Track your income and expenses.
*   **Mood Tracker**: Log how you feel with emojis.
*   **AI Assistant**: (Demo) Chat with an AI to help organize your life.

## Plans

*   **Free**: Basic access to all tools.
*   **Premium (R$10/mo)**: AI Chat, No Ads, Social features.

## Tech Stack

*   Vite
*   Vanilla JavaScript
*   Vanilla CSS (Glassmorphism Design)

## How to Run

1.  `npm install`
2.  `npm run dev`
